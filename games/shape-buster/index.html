Game Description

SHAPE-buster is a quick, colorful and satisfying clicker game where you pop the SHAPES before time runs out. Yeah, that's it. Simple, right?

But you've got a limited amount of time, and you need to click as many shapes as you can. The faster you pop them, well... the time will run out more faster :)
